# Welcome

mdv is a terminal-first markdown visualizer/editor.

> Mental model: left pane writes source, right pane previews render.

## Start Fast

- Open docs any time: `Cmd+,` (macOS) / `Ctrl+,` (Windows/Linux)
- Switch mode: `Shift+Tab` (macOS + Windows/Linux)
- Quit app: `Ctrl+Q` (macOS + Windows/Linux)

# Welcome

mdv is a note editor and previewer that runs in your terminal.

You do not need to know TUIs or Markdown to get started.
If you can type a file name and press `Enter`, you can use mdv.

> Mental model: one side is for typing, the other side is for reading the formatted result.

## What You Can Expect

- Start on the Home screen
- Type a file name or path
- Press `Enter` to open it
- Press `Ctrl+S` to save

## Safety + Help

- Open docs any time: `Cmd+,` (macOS) / `Ctrl+,` (Windows/Linux)
- Switch typing/preview focus: `Shift+Tab`
- Quit the app: `Ctrl+Q`

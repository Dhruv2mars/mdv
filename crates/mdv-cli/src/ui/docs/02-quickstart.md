# Quickstart

## Open or Create

- Home screen: type path, press `Enter`
- Existing file opens live
- Missing file starts empty and saves on first `Ctrl+S`

## Save + Reload

- Save: `Ctrl+S` (macOS + Windows/Linux)
- Reload from disk: `Ctrl+R` (macOS + Windows/Linux)
- Reopen docs from Home: `Cmd+,` / `Ctrl+,`

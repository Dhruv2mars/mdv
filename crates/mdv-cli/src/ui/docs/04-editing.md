# Keyboard Editing Basics

## Start Simple

You can begin with plain text.
Markdown is optional.

Useful Markdown basics:
- `# Heading` for a heading
- `- item` for a bullet list
- blank line for a new paragraph

## Typing + Undo

- Type to insert text
- `Tab` inserts four spaces
- Undo: `Ctrl+Z`
- Redo: `Ctrl+Y` (Windows/Linux) / `Cmd+Shift+Z` (macOS)

## Line Basics

- New line: `Enter`
- Delete left: `Backspace`
- Delete right: `Delete`
